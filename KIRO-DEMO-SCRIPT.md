# How I Used Kiro for CI/CD — Live Demo Script

---

## What is Kiro?

Kiro is an AI-powered IDE (Integrated Development Environment) by Amazon. It's like VS Code but with an AI agent built-in that can:
- Write code for you
- Run terminal commands
- Create files and configurations
- SSH into servers
- Deploy applications
- Generate documentation

Think of it as having a senior DevOps engineer sitting next to you, but it's AI.

## Audience Q&A Prep

**Q: Is Kiro free?**
A: Kiro has a free tier. It's built by Amazon and available to try.

**Q: Does it work for other languages/frameworks?**
A: Yes — Python, Java, Go, any language. It reads your project and adapts.

**Q: Can it access my AWS account?**
A: Only if you provide credentials. It uses the same AWS CLI you'd use manually.

**Q: Is it safe to give it credentials?**
A: Kiro warned me about credential exposure and helped secure them. But always rotate keys after sharing.

**Q: Can it replace DevOps engineers?**
A: It's a tool to accelerate DevOps work, not replace engineers. You still need to understand what's happening and make decisions.
